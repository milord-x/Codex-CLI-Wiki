# Example skills

This directory contains sample skills for Codex:

- `python-bugfix/` — a narrow workflow for safe Python bug fixing.
- `repo-docs/` — a skill for generating repository documentation from confirmed facts.

Both examples show the minimum useful pattern:

- a dedicated skill directory;
- a `SKILL.md` file;
- a clear purpose;
- step-by-step workflow;
- explicit constraints and validation.

Use these as templates, not as universal drop-in skills.
